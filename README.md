# Data Professional Survey Analysis (Power BI)

## Project Overview
This project analyzes survey responses from data professionals to understand career trends, salary distribution, programming language preferences, and job satisfaction. Power BI was used to create an interactive dashboard, with Excel serving as the data source.

The dashboard provides a high level view of demographic distribution, perceived difficulty of entering data roles, and satisfaction levels among professionals.

---

## Objectives
- Analyze demographic distribution of survey respondents
- Evaluate perceived difficulty of breaking into data roles
- Compare average salary across job titles
- Identify preferred programming languages
- Assess work life balance and salary satisfaction

---

## Dashboard Features
- Country wise distribution of survey participants
- Difficulty rating for entering data science and analyst roles
- Average salary comparison by job title
- Programming language preference analysis
- Satisfaction metrics for work life balance and salary

---

## Key Insights
- Python is the most commonly used programming language among data professionals
- Data Scientists and Data Engineers report higher average salaries
- A significant portion of respondents perceive entering data roles as moderately difficult
- Work life balance satisfaction scores are higher than salary satisfaction scores

---

## How to Use
1. Download the PowerBi.pbix file
2. Open the file using Power BI Desktop
3. If prompted, ensure the Excel file is correctly linked as the data source
4. Interact with the dashboard to explore insights
